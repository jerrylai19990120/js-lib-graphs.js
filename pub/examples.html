Graphs.js

A javascript library to easily generate statistical graphs for data analyptics and data visualization.


Using vanilla javascript, DOM manipulation and HTML canvas.

Histogram

Histogram for array data visualization. Customizing colors, width and height.

Pie Chart

Input an array into the api to obtain a colored pie chart.

Scatter Plot

Better visualization for statistical purpose.

Line Chart

It could be used to monitor stock price data, etc.

Stacked Histogram

For multiple groups of data comparison, a better visualization.

Area Graph

Visualizing the magnitude of the data set.

Pyramid Graph

Two groups of data comparison, for example, male and female age group.

Radar graph

Visualization of a set of data depend on distinct aspects.
